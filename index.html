Rascam-server

RasCam is a java server for controlling the Raspberry Pi camera board.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Rascam Server.

This project is designed to provide a simple json / http service for the Raspberry-Pi camera module and make it available to my IOS client application. The service will in future allow a username / password secured via https.
